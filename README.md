# Knightly

This repository contains the asset files and non-application source code for Knightly, an automatic chessboard product by Chessington. The board is able to detect its own state, move pieces, and detect moves by human players.

This repository contains SOLIDWORKS CAD files for the board, pieces, clock, and other components, STEP files created in KiCAD for the design of the printed circuit boards, Arduino microcontroller code for the detection of chess pieces, and our final project poster.

## Folder Structure

CAD files related to the board are available under [Solidworks CAD](<./Solidworks CAD/>). CAD files related to the detection PCBs are available under [PCB CAD](<./PCB CAD/>). Microcontroller code for the detection system is available under [STMDetection](./STMDetection/). Application code is available under [knightly-app](./knightly-app/) (separate repository).
